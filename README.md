# Sistema de Suporte 💜

Sistema simples de suporte de chamados desenvolvido em PHP + MySQL.

## 📌 Funcionalidades

- Criar chamados
- Listar chamados
- Editar chamados
- Excluir chamados

## 🧠 Tecnologias utilizadas

- PHP
- MySQL
- HTML
- CSS
- XAMPP

## ⚙️ Como executar

1. Colocar o projeto dentro da pasta `htdocs` do XAMPP
2. Iniciar Apache e MySQL
3. Criar o banco de dados no MySQL
4. Acessar no navegador:
 ```
http://localhost/SistemaDeSuporte
```

## 🗄️ Banco de dados

Tabela: chamados
Campos:
- id (INT, PK, AUTO_INCREMENT)
- titulo (VARCHAR)
- descricao (TEXT)
- status (VARCHAR)
